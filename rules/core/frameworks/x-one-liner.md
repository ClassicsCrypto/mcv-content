# X One-Liner Framework

- **Bucket:** one-liner
- **Length:** under 100 characters, including spaces
- **Primary job:** fast scroll-stop, quotability, visual captioning, live pings, identity signals
- **Use for:** sharp takes, punchy quotes, provocative questions, visual-first captions, recurring
  ritual beats
- **Do not use for:** explanations, multi-step ideas, unsupported hype, fake mystery

## When to choose this bucket

Use a one-liner when the post needs to do exactly one sharp job: name a moment, label a behavior,
make a visual feel important, set up a quotable contrast, prompt a quick reply, or signal a live
state. Image/video-heavy posts usually use this bucket or `short-standard`.

## Core shape

A one-liner works when it feels obvious after reading:

```text
[specific noun or tension] + [short judgment, contrast, or action]
```

Strong one-liners usually carry one of:

- a real, specific noun (a named event, place, artifact);
- a contrast (`[specific thing] > [generic claim]`);
- a state change (`Live now`, `Verified`, `The route is open`);
- a mood that the attached visual proves.

## Archetype fit

Best for recurring-ritual, visual-showcase, conversational-reply, single-question engagement, and
one-quotable-truth thesis posts. Use a teaser only when there is a real, approved payoff.

## Anti-patterns (fail the draft)

- Vague `soon` / `incoming` / `big things` with no approved payoff.
- A line any account in the space could post unchanged (substitutable — see
  `rule.core.voice-register`).
- A question with no value attached.
- A line that needs a second sentence to make sense.

## Writer checklist

- Under 100 characters including spaces.
- One idea only; a specific noun appears early.
- No hashtags (Twitter/X); no mid-sentence em dash (`rule.core.formatting`).
- If visual-heavy, the line names what the viewer should notice.
- If text-only, the line is quotable without context.

## Example (brand: Acme Cosmos)

```text
The Acme Cosmos beta did not get announced. It got finished.
```
